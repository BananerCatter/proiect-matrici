Algoritmul de Fill

Ce este si ce face acest algoritm? Acest algoritm este un algoritm ce se foloseste de obicei atunci cand avem de lucru cu o matrice. Ce facest acest algoritm? Are multe aplicatii, dar una din cele mai practice este atunci cand determinam numarul elementelor dintr-o matrice care respecta o anumita conditie.
