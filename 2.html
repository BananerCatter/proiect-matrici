Matrici pătratice

Definiție:

Matricile pătratice sunt matrici care au acelasi număr de linii și coloane:

int a[n][n];

Urmatoarea matrice ar avea dimesiunea de n2, justificând astfel numele de matrice pătratică

Într-o matrice pătratică se disting o categorie specială de elemente, diagonalele. Un element al matricei aparține sau nu diagonalelor sau zonelor delimitate de acestea dacă respectă anumite reguli, ce depind de indicii elementului. În cele ce urmează, pentru un element al matricei vom nota cu i indicele de linie și cu j indicele de coloană.

Diagonala principală

Pe diagonala principală, i == j Elementele sunt parcurse astfel:

for(int i = 0 ; i < n ; i ++)
{
    // A[i][i]
}

Pe diagonala secundară, daca matricea e indexată de la 0, i + j == n - 1, iar daca e indexată de la 1, i + j == n + 1

//indexare de la 0
for(int i = 0 ; i < n ; i ++)
{
    // A[i][n - 1 - i]
}
                
//indexare de la 1
for(int i = 0 ; i < n ; i ++)
{
    // A[i][n + 1 - i]
}

Elementele delimitate de diagonala principală:

  • Sub diagonală: i < j
  • Peste diagonală: i > j

Elementele delimitate de diagonala secundară

    Indexare de la 0:

  • Sub diagonală: i + j < n - 1
  • Peste diagonală: i + j > n - 1

    Indexare de la 1:

  • Sub diagonală: i + j < n + 1
  • Peste diagonală: i + j > n + 1

  • Algoritmi avansați
